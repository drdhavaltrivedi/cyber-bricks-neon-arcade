COMMON QUERIES

FREQUENTLY ASKED PROTOCOLS

Direct answers about Cyber-Bricks installation, compatibility, permissions, and security protocols.

What makes Cyber-Bricks unique as a browser extension?

Cyber-Bricks is a zero-dependency breakout game designed to launch instantly from your toolbar without opening new tabs. It uses programmatically synthesized Web Audio (rather than loading heavy media files), making it extremely fast, lightweight, and offline-compatible.

How do I install Cyber-Bricks on Chrome, Edge, or Firefox?

You can download it from the official Chrome Web Store, Firefox Add-ons, or Edge Add-ons directories. For manual developer installation, clone the repository from GitHub, enable 'Developer Mode' in your browser's extension settings, and load the unpacked extension folder.

Does Cyber-Bricks collect or track my data?

No. Cyber-Bricks operates under a strict Zero Data Collection policy. The game runs 100% locally on your computer. Settings and high scores are saved locally using standard sandboxed storage, and no remote network requests are ever made.

What permissions does the extension require?

The extension only requests the 'storage' permission, which is required to save your configuration settings and high scores locally in your browser sandboxed storage. No other sensitive browser data is accessed.
